Algorithm of Merge Sort



Best Case Complexity - 
1)The best case time complexity of quick sort is O(Nlog(N))
2)The best case of quick sort is when we will select pivot as a mean
element.

Worst Case Complexity -
1)The worst case time complexity of quick sort is O(N^2)
2)This will happen when our array will be sorted and we select 
smallest or largest indexed element as pivot.

            
Average Case Complexity - 
1)The average case time complexity of quick sort isO(Nlog(N))
2)The overall average case for the quick sort is this which we will
get by taking average of all complexities.

Algorithm of Linear Search



Best Case Complexity - 
1)The element being searched could be found in the first position.
2)In this case, the search ends with a single successful comparison.
3)Thus, in the best-case scenario, the linear search algorithm performs O(1) operations.

Worst Case Complexity -
1)The element being searched may be at the last position in the array or not at all.
2)In the first case, the search succeeds in ‘n’ comparisons.
3)In the next case, the search fails after ‘n’ comparisons.
4)Thus, in the worst-case scenario, the linear search algorithm performs O(n) operations.

Average Case Complexity - 
When the element to be searched is in the middle of the array, the average case of the
Linear Search Algorithm is O(n).
